# よく使うコマンド

このページではよく使うAsteriskのコマンドをまとめています

## CLIに入る

AsteriskのCLIに入るには以下のコマンドを実行します。
CLIに入ると、Asteriskの状態を確認したり、コマンドを実行したりすることができます。
Control + CでCLIから抜けることができます。

```bash
sudo asterisk -rvvv
```

### 設定のリロード
Asteriskの設定を変更した後、変更を反映させるために設定のリロードが必要です。
以下のコマンドをCLIで実行することで、設定をリロードすることができます。

```bash
pjsip reload
dialplan reload
```

CLIの外で設定のリロードを行う場合は、以下のコマンドを実行します。

```bash
sudo asterisk -rx "pjsip reload"
sudo asterisk -rx "dialplan reload"
```

## 内線の状態を確認する

AsteriskのCLIで以下のコマンドを実行することで、内線の状態を確認することができます。

```bash
pjsip show endpoints
pjsip show contacts
pjsip show aors
dialplan show コンテキスト名
```
CLIの外で内線の状態を確認する場合は、以下のコマンドを実行します。

```bash
sudo asterisk -rx "pjsip show endpoints"
sudo asterisk -rx "pjsip show contacts"
sudo asterisk -rx "pjsip show aors"
sudo asterisk -rx "dialplan show コンテキスト名"
